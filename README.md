# ControlCash

ControlCash es un sistema por ahora local de gestión de finanzas personales que permite registrar, filtrar y visualizar tus transacciones de manera sencilla.
Permite llevar en tiempo real tus movimientos comunes del dia a dia, esos que tal vez son "gastos hormiga", tanto como los mas importantes (gastos mensuales) y asi poder tener un mayor control de tu economía diaria.

## Características

- Registro manual en el sistema de ingresos y gastos con fecha, categoría y descripción.
- Registro automático en el sistema a través de un bot en telegram.
- Filtrado de transacciones por fecha y categoría.  
- Visualización de las 20 transacciones más recientes.  
- Integración con telegram API, para vincular y desvincular el usuario del sistema con el usuario de telegram.
- Gestión de categorías personalizables.
- Notificaciones en tiempo real al cargar un nuevo registro desde Telegram.
